# IT Support Portal — Documentation

Welcome to the IT Support Portal documentation. This `docs/` folder is intended for GitHub Pages or repository documentation viewers.

Available guides:

- [Project Structure](PROJECT_STRUCTURE.md) — Overview of repository layout and components.
- [Deployment Guide](DEPLOYMENT_GUIDE.md) — Production deployment instructions (Proxmox/Docker/etc.).
- [Implementation Summary](IMPLEMENTATION_SUMMARY.md) — High-level description of implemented features and architecture.

How to publish the docs on GitHub Pages:

1. In repository Settings → Pages, select the `docs/` folder on the `main` branch.
2. Optionally enable a custom domain and HTTPS.

Contributing
------------

If you update docs, please edit the files in this folder and open a pull request. Keep content concise and refer to source files for code-level details.
